What is Bowstal - Craft?

BowStalCraft is a pre-configured minecraft server, that you can use in your project

What does BowStal - Craft has?

It has a lot of plugins pre-innstalled and properly configured on minecraft versions --- and ---. Additionaly it has a lot of built by hand maps. That you can use on your server.

How you can use it?

You can use our configured server as the starter setup. If you want you can use all our maps in production, it is really up to you what to do!
